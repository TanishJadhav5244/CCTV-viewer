AI Pipeline

Four-stage deep learning pipeline runs automatically once you start processing.

1
Frame Extraction

OpenCV reads frames from any source — file, RTSP, or webcam.

→
2
YOLOv8 Segmentation

Detects and segments objects: people, vehicles, objects with bounding boxes.

→
3
Attribute Detection

CLIP zero-shot classifies clothing color, gender cues, and accessories per person.

→
4
CLIP Embedding + pgvector

Each crop → 512-dim semantic vector stored in PostgreSQL for instant text search.

  • RTSP streams are supported (e.g. IP cameras)
  • Larger videos will take longer — FPS shown during processing
  • Switch to Detections tab while processing to see results live
